3. **신규 독립 스크립트 3개 작성** (`model-preparation/` 아래):

   - 각 파일에 `effective_cpus()`(cgroup) 복사 → 자립. 원본 `build_artifact.py` 미수정.
   - `py_compile` 통과 확인. 실제 dry-run은 아직 안 함.

4. **문서 갱신.** 3개 모델 README + 상위 `model-preparation/README.md`를 신규 스크립트에 맞춤.
