## 한 일 (시간순)

1. **상태 복구.** `session-handoff` 메모리에서 furiosa-student-kit 작업 맥락 복원.
   사용자 확인: git push·VS Code 정비는 이미 완료(사용자 직접).

2. **모델별 스크립트 방침 논의.** 내 최초 권고는 "공용 1개 유지"였으나, 사용자가
   "별도 파일로 만들되 원본은 건드리지 말자"로 결정 → 그 방향으로 실행.

3. **신규 독립 스크립트 3개 작성** (`model-preparation/` 아래):
   - `Qwen3/build_qwen3.py` — 프리셋 완비, 모델 ID 하드코딩, `--fix-append` 제거.
   - `Llama/build_llama.py` — gated 대비 HF 토큰 미검출 경고 추가.
   - `Gemma4_try/build_gemma_TEMPLATE.py` — `GEMMA_*` 버킷 4종 빈 뼈대 + 미완 시 SystemExit.
   - 각 파일에 `effective_cpus()`(cgroup) 복사 → 자립. 원본 `build_artifact.py` 미수정.
   - `py_compile` 통과 확인. 실제 dry-run은 아직 안 함.

4. **문서 갱신.** 3개 모델 README + 상위 `model-preparation/README.md`를 신규 스크립트에 맞춤.

5. **협업 파일 규칙 수립(사용자 지시).**
   - `docs/handoff/claudex-handoff.md` = Codex 전달용(내가 처음엔 dated 파일로 만들었다가
     `claudex-handoff.md` 단일 롤링 파일로 통합).

   - 그리고 이 두 규칙을 `AGENTS.md`에 "에이전트 간 협업 규칙" 섹션으로 추가.
